# Tic-Tac-toe
This project is a classic Tic Tac Toe (O-X) game implemented in Python, played on a 3Ã—3 grid. The game pits the user against the computer, where each player alternately marks a cell with either X or O. The objective is to align three of your symbols horizontally, vertically, or diagonally before your opponent does.
The computer plays with the 'o' and the user plays with 'x'.
The player who can fill their respective element in a horizontal row,vertical row or a diagonal completely will win the game.
The computer will get trace your moves and it makes it moves with according to that.
